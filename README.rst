
MooTools Konami Code Event
==========================

Adds a konami code event to all elements.

The Konami Code
---------------

+----+----+------+------+------+-------+------+-------+---+---+ 
| up | up | down | down | left | right | left | right | b | a |
+----+----+------+------+------+-------+------+-------+---+---+ 

Usage
-----

::
  
  window.addEvent('konami', function(){
    // ..
  });
